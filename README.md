# Moonfox Legal & Info Pages

Static website for **Moonfox / 狐问六爻**, hosted on GitHub Pages.

## Live URLs

After deployment, the site is available at:
`https://guowuche.github.io/moonfox-legal/`

| Page | URL | Purpose |
| --- | --- | --- |
| Home / About | `index.html` | App introduction & landing page |
| Privacy Policy | `privacy.html` | Used in Google Play "Privacy policy" field |
| Terms & Disclaimer | `terms.html` | User agreement & I Ching disclaimer |
| Account Deletion | `delete-account.html` | Used in Google Play "Account deletion URL" field |

## Files

- `index.html` — Landing page introducing Moonfox (English / 简体中文)
- `privacy.html` — Full Privacy Policy (English / 简体中文)
- `terms.html` — Terms of Service & Disclaimer (English / 简体中文)
- `delete-account.html` — Account & data deletion request page (English / 简体中文)

## Deployment (GitHub Pages)

1. Push these files to your `moonfox-legal` GitHub repository.
2. In repository **Settings → Pages**, set:
   - Source: `Deploy from a branch`
   - Branch: `main` / `(root)`
3. Wait ~1 minute, then visit `https://guowuche.github.io/moonfox-legal/`.

## Notes

- All pages are bilingual (English default, switchable to 简体中文).
- Auto-detects browser language (zh-* → 中文).
